ВНИМАНИЕ! На момент загрузки проекта, все находящиеся в нем тесты работают! По прохождению времени буду стараться их проверять и в случае присутствия ошибок исправлять.

Описание тестов:
1. ВВОД ДАННЫХ. 
Просто тестирует работоспособность поиска и факт выдачи результата.

2. DATA DRIVEN/УСЛОВНЫЕ ОПЕРАТОРЫ. 
Тестирование валидации. Проверяет 3 варианта недопустимого для поиска значений (ссылка; спец. символы; слишком большое значение). Также проверяет выданный поиском результат (допустим здесь он задокументирован и известен заранее).

3. ЦИКЛЫ
На стартовой странице есть большой блок со всеми представленными марками авто на сайте. Робот открывает каждую из представленных марок и проверяет работоспособность ссылки, название марки и соответствует ли результат изначально запрошенной марке авто. Тест не закончится пока робот не проверит каждую марку.
В работе использовал только For, так как для While не нашел функциональности на сайте.
